# Web PPT Creator - 网页PPT创建器

网页PPT生成器 - 8步对话式创建流程，支持10种主题、9种动画效果，生成交互式HTML演示文稿

通过对话引导创建网页版PPT的Skill工具。

## 功能特点

- **8步对话式创建**：从大纲到生成，全程对话式交互
- **大纲讨论**：AI帮助你完善PPT大纲，推荐主题风格和模板
- **模板选择**：12种预设模板可选
- **内容描述**：自由描述每页内容，AI生成结构化幻灯片
- **主题风格**：10种主题风格（科技蓝、商务黑、清新绿、渐变紫等）
- **动画效果**：9种动画效果（淡入淡出、滑动、缩放、3D翻转等）
- **交互功能**：键盘导航、鼠标点击、触摸滑动、激光笔、全屏等
- **单文件HTML**：导出为独立HTML文件，双击即可本地预览

## 使用方法

### 1. 启动Skill

在CodeFuse中调用此skill：
```
/web-ppt-creator
```

### 2. 8步创建流程

| 步骤 | 说明 |
|------|------|
| 步骤1 | 基本信息 - 输入标题、副标题、使用场景 |
| 步骤2 | 大纲设计 - AI生成大纲，用户确认或修改 |
| 步骤3 | 页面详情 - 为每页选择布局并填写内容 |
| 步骤4 | 主题风格 - 选择10种主题之一 |
| 步骤5 | 动画效果 - 选择9种动画之一 |
| 步骤6 | 交互功能 - 选择支持的交互方式 |
| 步骤7 | 预览确认 - 确认所有配置 |
| 步骤8 | 生成导出 - 生成HTML文件 |

### 3. 命令行使用

```bash
# 使用配置文件生成
python scripts/generate_ppt.py --config config.json --output my-ppt.html

# 快速生成
python scripts/generate_ppt.py --title "我的PPT" --theme 科技蓝 --animation 淡入淡出 --output presentation.html

# 查看所有选项
python scripts/generate_ppt.py --list-themes
python scripts/generate_ppt.py --list-animations
python scripts/generate_ppt.py --list-templates
```

### 4. 交互式创建

```bash
python scripts/interactive_creator.py
```

启动后按提示一步步完成PPT创建。

## 主题风格选项（10种）

1. **科技蓝** - 蓝色渐变，现代科技感
2. **商务黑** - 黑色主调，专业商务
3. **清新绿** - 绿色系，自然清新，适合教育主题
4. **渐变紫** - 紫色渐变，梦幻优雅
5. **渐变橙** - 橙色渐变，活力温暖
6. **毛玻璃** - 磨砂玻璃效果，时尚现代
7. **中国红** - 红色系，喜庆大气
8. **深海蓝** - 深蓝渐变，深邃沉稳
9. **极简白** - 白色主调，简洁干净
10. **赛博朋克** - 霓虹色，高对比，适合创意主题

## 动画效果选项（9种）

1. **无动画** - 静态切换
2. **淡入淡出** - Fade效果，经典柔和
3. **滑动进入** - Slide效果，流畅自然
4. **缩放效果** - Zoom效果，视觉冲击
5. **组合动画** - 多种效果混合
6. **3D翻转** - Flip效果，科技感强
7. **立方体** - Cube效果，立体切换
8. **覆盖** - Cover效果，新页面覆盖
9. **揭开** - Uncover效果，像揭开幕布

## 交互功能选项

- **键盘导航** - 左右方向键切换
- **鼠标点击** - 点击屏幕边缘翻页
- **触摸滑动** - 支持手机/平板滑动
- **进度条** - 底部显示当前进度
- **缩略图导航** - 显示所有页面缩略图
- **激光笔** - 按住鼠标显示激光点
- **全屏模式** - 支持全屏演示
