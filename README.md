# Web PPT Creator - 网页PPT创建器

通过对话引导创建网页版PPT的Skill工具。

## 功能特点

- 大纲讨论：AI帮助你完善PPT大纲，推荐主题风格和模板
- 模板选择：多种预设模板可选
- 内容描述：自由描述每页内容，AI生成结构化幻灯片
- 主题风格：多种主题风格（科技蓝、商务黑、清新绿等）
- 动画效果：多种动画效果可选
- 交互功能：键盘导航、鼠标点击、进度指示器等
- Playground预览：生成后可预览效果
- 导出HTML：导出为独立文件夹，双击即可本地预览

web-ppt-creator/

## 使用方法

### 1. 启动Skill

在CodeFuse中调用此skill：
```
/web-ppt-creator
```

或通过Skill工具调用：
```
Skill: web-ppt-creator
```

### 2. 完整流程

| 阶段 | 说明 |
|------|------|
| Phase 0 | 大纲讨论 - AI分析主题，推荐风格和模板 |
| Phase 1 | 模板选择 - 选择基础模板类型 |
| Phase 2 | 内容描述 - 描述每页幻灯片内容 |
| Phase 3 | 主题风格 - 选择主题颜色 |
| Phase 4 | 动画效果 - 选择切换动画 |
| Phase 5 | 交互功能 - 选择交互方式 |
| Phase 6 | 预览调整 - Playground预览效果 |
| Phase 7 | 导出HTML - 下载完整HTML文件夹 |

### 3. 导出文件结构

```
ppt-output/
├── index.html          # 主文件
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── main.js         # 交互脚本
├── assets/
│   └── images/         # 图片资源
└── README.md           # 打开说明
```

**使用说明：** 双击 `index.html` 即可在浏览器中预览，无需服务器。

## 对话示例

```
用户：我要做一个产品发布会的网页PPT，主题是新产品发布
AI：好的！让我帮你分析一下大纲...
    建议大纲包含：开场、产品介绍、核心功能演示、定价策略、总结
    推荐风格：科技蓝/渐变紫
    适合模板：标题页+两栏布局+图片展示
    (可上传参考图片/网页链接)

用户：好的，我上传一个参考图片
AI：收到！请继续选择模板...
```

## 主题风格选项

1. 科技蓝 - 蓝色渐变，现代科技感
2. 商务黑 - 黑色主调，专业商务
3. 清新绿 - 绿色系，自然清新
4. 渐变紫 - 紫色渐变，梦幻优雅
5. 渐变橙 - 橙色渐变，活力温暖
6. 毛玻璃 - 磨砂玻璃效果，时尚现代
7. 自定义 - 描述你喜欢的颜色风格

## 动画效果选项

1. 无动画 - 静态切换
2. 淡入淡出 - Fade效果
3. 滑动进入 - Slide效果
4. 缩放效果 - Zoom效果
5. 组合动画 - 多种效果组合
6. 自定义描述

## 交互功能选项

1. 键盘导航 - 左右方向键切换
2. 鼠标点击 - 点击切换
3. 进度指示器 - 显示当前页码/总页数
4. 放大镜模式 - 点击查看大图
5. 激光笔效果 - 鼠标跟随高亮
6. 全屏模式 - 支持全屏演示

## 模板选项

1. 空白模板 - 从零开始
2. 标题页 - 适合开场
3. 两栏布局 - 文字+图片
4. 图片展示 - 适合作品集
5. 代码演示 - 适合技术分享
6. 自定义描述

## 技术栈

- HTML5 + CSS3
- JavaScript (原生)
- 无需后端服务器
