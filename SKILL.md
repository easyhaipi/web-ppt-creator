# 网页PPT创建器 (Web PPT Creator)

## 概述

**不会设计？不懂代码？没问题！**

网页PPT创建器是一个**零门槛、全对话式**的PPT制作工具。你只需要像聊天一样回答AI的问题，8步之内就能生成一份专业、美观、可交互的网页版PPT。

### 为什么选择网页PPT创建器？

| 传统PPT工具 | 网页PPT创建器 |
|-----------|--------------|
| 需要设计基础，纠结排版配色 | **AI全程引导，智能推荐最佳方案** |
| 软件安装繁琐，版本兼容问题 | **输出纯HTML，双击即用，无需安装** |
| 静态展示，缺乏互动 | **支持键盘/鼠标/触摸交互，演示更生动** |
| 换电脑需拷贝文件 | **一个HTML文件，随时随地打开** |
| 模板千篇一律 | **10种主题+9种动画，打造独特风格** |

### 适用场景

- 💼 **工作汇报** - 周报、月报、季度总结、项目复盘
- 🚀 **产品发布** - 新品发布、功能介绍、路演展示

🗣️ **对话式创建** - 像聊天一样做PPT，AI问你答，无需手动排版

## 快速开始（3秒上手）

### 方式1: 交互式对话创建（强烈推荐 ⭐）

```bash
python scripts/interactive_creator.py
```

然后**跟着AI的提问一步步回答**，就像填问卷一样简单！

**典型使用流程：**
1. 告诉AI你要做什么PPT（标题+场景）

# 常见问题

**Q: 我不会设计，能做出来好看的PPT吗？**
A: 完全可以！这个工具就是为非设计人员打造的。AI会全程引导你，自动处理排版、配色、动画等设计细节，你只需要专注内容即可。

**Q: 导出的是什么格式？需要安装软件吗？**
A: 导出为 **.html** 格式文件，**无需安装任何软件**，双击即可在浏览器中打开。Windows、Mac、Linux、手机、平板都能正常显示。

**Q: 制作一份PPT需要多长时间？**
A: 使用交互式创建，通常 **10-15分钟** 就能完成一份专业PPT。比传统PPT软件快5-10倍，而且质量更有保障。

**Q: 可以在手机上演示吗？**
A: 可以！生成的HTML文件支持触摸滑动翻页，完美适配手机和平板。你还可以开启"缩略图导航"功能，方便在移动设备上快速跳转。

**Q: 如何修改已生成的PPT？**
A: 两种方式：
   1. 重新运行 `python scripts/interactive_creator.py`，AI会记住之前的配置，你可以在此基础上修改
   2. 直接编辑生成的HTML文件（适合懂代码的用户）

**Q: 支持插入图片吗？**
A: 支持！在内容描述阶段可以上传参考图片。如果需要插入大量图片，建议生成后在HTML中手动添加 `<img>` 标签。

**Q: 有哪些主题和动画可选？**
A: 10种主题（科技蓝、商务黑、赛博朋克、清新绿等），9种动画（淡入淡出、3D翻转、立方体等）。运行 `python scripts/generate_ppt.py --list-themes` 查看完整列表。

**Q: 交互式创建过程中断怎么办？**
A: 程序会自动保存草稿到 `ppt-output/interrupted_draft.json`，重新运行脚本会提示是否继续上次的进度。

**Q: 这个工具收费吗？**
A: 完全免费！开源工具，本地运行，无需联网，数据安全。

**Q: 生成的PPT可以商用吗？**
A: 可以！生成的HTML文件完全属于你，可自由使用、修改、分发，无任何限制。
